# Calculadora de Optimización Fiscal - DERTOGEST, S.L.

Herramienta interna para el cálculo de incentivos fiscales basados en el **Art. 39.7 LIS** (Tax Lease).

## Funcionalidades:
- [cite_start]Cálculo de rentabilidad neta (20%) para inversores[cite: 29].
- [cite_start]Aplicación de límites de cuota (25% - 50%) según normativa vigente[cite: 13, 14, 39].
- [cite_start]Diferenciación entre Persona Física (IRPF) y Sociedad (IS)[cite: 11, 28].

## Seguridad:
[cite_start]Cada operación debe estar respaldada por su correspondiente **Informe Motivado** y **Seguro de Contingencia** gestionado por DERTOGEST[cite: 20, 24, 51].
